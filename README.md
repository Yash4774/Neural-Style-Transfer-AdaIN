# вҡҷпёҸ Installation

Clone the repository

```bash

```

Move into project

```bash
cd StyleForge-AI
```

Install dependencies

```bash

```

```bash

```


# рҹ§Ә Training

Train the model

```bash
python train.py \
--content_dir DATASET \
--style_dir STYLESET \
--epochs 180
```

Resume Training

```bash
python train.py \
--resume \
--decoder_path decoder_180.pth \
--optimizer_path optimizer_180.pth
```
