# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/StyleForge-AI.git
```

Move into project

```bash
cd StyleForge-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

---

# 🧪 Training

Train the model

```bash
python train.py \
--content_dir DATASET \
--style_dir STYLESET \
--epochs 205
```

Resume Training

```bash
python train.py \
--resume \
--decoder_path decoder_205.pth \
--optimizer_path optimizer_205.pth
```
